#  Digital Village Governance System

A web-based application developed to digitize and streamline village-level governance services.

This system enables efficient management of citizen records, service requests, government schemes, and administrative operations through a secure and centralized platform.

---

##  Tech Stack

###  Backend
- Java
- Spring Boot
- Spring Security
- Spring Data JPA
- Hibernate
- MySQL
- Maven

###  Frontend (if applicable)
- React / HTML / CSS / JavaScript

---

##  Key Features

- Citizen Registration & Management
- Admin Login & Role-Based Access Control
- Service Request Management
- Government Scheme Tracking
- Complaint Registration & Status Tracking
- Secure Authentication & Authorization
- RESTful API Architecture
- MySQL Database Integration

---

##  Database Design

The system manages structured relational data for:

- Citizens
- Administrators
- Service Requests
- Complaints
- Government Schemes
- Village Records

---

## ⚙️ How To Run The Project Locally

###  Run Backend

1. Open terminal
2. Navigate to project folder:

cd DigitalVillageGovernanceSystem

3. Run the application:

mvnw.cmd spring-boot:run

4. Backend will start at:

http://localhost:8080

---

###  Run Frontend (If Available)

cd frontend  
npm install  
npm start  

Open browser:

http://localhost:3000

## 🎯 System Modules

###  Citizen Module
- Register citizen
- View available services
- Apply for government schemes
- Track complaint status

###  Admin Module
- Manage citizen records
- Approve or reject service requests
- Update complaint status
- Add and manage government schemes
